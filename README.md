# Крестики-нолики (Tic-Tac-Toe)

Сетевая игра «Крестики-нолики» для двух игроков в локальной сети.

## Краткое описание

Клиент-серверное приложение, позволяющее двум игрокам играть в крестики-нолики по сети. Сервер управляет игровым процессом, клиенты отображают поле и принимают ходы. Реализована таблица лидеров с сохранением результатов.

## Основной функционал

- Подключение к серверу по IP и порту
- Игровое поле 3×3 с отображением X и O
- Поочерёдные ходы двух игроков
- Определение победителя или ничьей
- Таблица лидеров
- Автоматический перезапуск игры после завершения

## Поддерживаемые платформы

- Windows 10/11 (x64)
- .NET Framework 4.7.2

## Инструкция по работе

### Запуск сервера
1. Запустите `TicTacToe.Server.exe`
2. Сервер запустится на порту 8888

### Запуск клиентов
1. Запустите `TicTacToe.Client.exe` два раза
2. В каждом окне нажмите «Подключиться» (IP: 127.0.0.1)
3. Первый подключившийся играет за X (ходит первым)
4. Второй подключившийся играет за O

### Игровой процесс
- Нажимайте на клетки поля для совершения хода
- Побеждает тот, кто соберёт 3 своих символа в ряд
- После завершения игра начнётся заново автоматически

## Сборка из исходников

1. Откройте `TicTacToe.sln` в Visual Studio 2019+
2. Установите NuGet-пакеты: Newtonsoft.Json, NUnit, NUnit3TestAdapter
3. Соберите решение (Ctrl+Shift+B)
